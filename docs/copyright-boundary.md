# 版权边界说明

本仓库是面向武汉大学数字取证课程开卷考试复习的个人快查大纲项目。

为降低版权风险，本仓库只保留索引式大纲、关键名词和印刷页码，不包含教材正文、教材提取数据、课后题或答案。

## 不进入仓库的内容

- 教材 PDF、扫描件或电子书文件
- OCR / MinerU / 全文检索数据库等教材提取产物
- 教材原文长段落、图片、表格、例题和代码
- 课后题和课后题答案
- 未公开课程资料、考试题、教师课件

## 可以进入仓库的内容

- 章节级和知识点级索引
- 用于定位教材内容的印刷页码
- 自己编写的生成脚本
- 自己整理的简短复习提示

页码引用只用于帮助合法持有教材的学习者快速翻书定位。
